# Offertly

Offertly är en SaaS-produkt för hantverkare som vill skapa professionella offerter snabbt.

## Fas 1
Produktionsgrund med Streamlit.

## Starta appen

pip install -r requirements.txt
streamlit run app.py

## Kommande faser

1. Produktionsgrund
2. Supabase login
3. Stripe betalning
4. AI-offert + PDF
5. Kundportal / signering
